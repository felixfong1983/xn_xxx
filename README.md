小牛Admin v2.0.0
===============

> 运行环境要求PHP >= 8.0.0

## 框架介绍

* 本后台框架基于thinkphp8 + layui开发
* thinkphp8开发手册： https://doc.thinkphp.cn/v8_0/preface.html
* layui官方文档：https://layui.dev/docs/2/

## 安装

~~~
手动创建数据库，导入根目录下的db.sql，修改根目录下的.env文件配置数据库信息
~~~

运行目录
~~~
public
~~~
